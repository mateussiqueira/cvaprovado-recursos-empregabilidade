# Template de vaga verificada

Use este modelo para publicar uma vaga com informacoes minimas de seguranca e clareza. A ideia e reduzir spam, vaga falsa e candidatura perdida.

## Informacoes obrigatorias

```text
Empresa:
Site oficial da empresa:
Pessoa recrutadora ou contato responsavel:
LinkedIn/site do recrutador:
Cargo:
Senioridade:
Modelo: remoto / hibrido / presencial
Localidade:
Tipo de contrato: CLT / PJ / estagio / freelancer / temporario
Faixa salarial ou motivo para nao informar:
Beneficios principais:
Link oficial de candidatura:
Prazo estimado do processo:
```

## Descricao da vaga

```text
Contexto do time:
Principais responsabilidades:
Requisitos obrigatorios:
Requisitos desejaveis:
Ferramentas ou tecnologias usadas:
Como sera avaliada a candidatura:
Etapas do processo:
```

## Regras para comunidade

- Nao pedir documentos pessoais por DM.
- Nao pedir pagamento, curso, taxa, equipamento ou deposito para participar do processo.
- Nao publicar vaga sem empresa identificavel.
- Nao esconder modelo de trabalho quando presencial/hibrido for obrigatorio.
- Nao prometer contratacao imediata.
- Atualizar a vaga quando fechar.

## Mensagem pronta para recrutador

```text
Oi, pessoal. Sou recrutador(a) da [empresa] e estou divulgando uma oportunidade de [cargo].

Preenchi o template da comunidade para deixar a vaga verificavel: empresa, link oficial, modelo de trabalho, faixa/criterio de salario e etapas do processo.

Quem tiver duvidas pode perguntar nos comentarios. Para dados pessoais e envio de curriculo, usem apenas o link oficial de candidatura.
```
