# Como pedir feedback de CV sem se expor

Pedir feedback publico pode ajudar muito, mas tambem pode expor dados pessoais. Use este guia para receber ajuda com mais seguranca.

## Prepare o contexto

Antes de postar, escreva:

```text
Cargo-alvo:
Senioridade:
Tipo de empresa/vaga:
Pais ou mercado:
Principal duvida:
```

Exemplo:

```text
Cargo-alvo: Desenvolvedor Front-end Junior
Senioridade: Junior
Tipo de vaga: produto SaaS ou agencia
Mercado: Brasil/remoto
Principal duvida: meu curriculo parece generico demais?
```

## Anonimize o curriculo

Remova:

- nome completo;
- foto;
- telefone;
- e-mail;
- CPF/RG;
- endereco completo;
- links privados;
- nome de empresa sensivel;
- nome de colegas, gestores ou clientes.

Mantenha:

- cargo-alvo;
- responsabilidades;
- resultados;
- ferramentas;
- formacao geral;
- projetos relevantes.

## Faça uma pergunta especifica

Pergunta ruim:

```text
O que acharam do meu curriculo?
```

Pergunta melhor:

```text
Estou buscando vaga de Analista de Dados Junior. O primeiro terco do curriculo deixa isso claro? Minhas experiencias mostram resultados ou parecem apenas tarefas?
```

## Separe opiniao de criterio

Ao receber feedback, pergunte:

- Isso melhora aderencia com a vaga?
- Isso facilita leitura para recrutador?
- Isso reduz risco de ATS?
- Isso remove ruido ou so muda gosto visual?

## Depois do feedback

1. Escolha ate tres mudancas.
2. Ajuste para uma vaga especifica.
3. Salve uma versao nova do arquivo.
4. Teste se o texto pode ser selecionado no PDF.
5. Aplique e registre o resultado.

## Regra de ouro

Feedback publico deve melhorar clareza, nao expor sua vida. Se a pessoa precisa de seus documentos ou dados privados para avaliar um curriculo, provavelmente nao e o canal certo.
